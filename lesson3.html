Trigonometry Notes

Right-Angled Triangle Basics

In a right-angled triangle, the sides have special names:

  • Hypotenuse: The longest side, opposite the right angle.
  • Opposite: The side opposite to the angle we are considering (often labeled as θ).
  • Adjacent: The side next to the angle we are considering (not the hypotenuse).

Pythagoras Theorem

This theorem tells us the relationship between the sides of a right-angled triangle:

(Hypotenuse)2 = (Opposite)2 + (Adjacent)2

Example Using Pythagoras Theorem

Consider a right-angled triangle with sides 3 and 4. Let's find the hypotenuse (x).

Using Pythagoras Theorem, we can calculate the length of the hypotenuse:

x2 = 32 + 42

x2 = 9 + 16

x2 = 25

x = √25 = 5

Using Sine, Cosine, and Tangents

These are special ratios of the sides of a right-angled triangle related to its angles:

  • Sine (Sin θ): Opposite / Hypotenuse
  • Cosine (Cos θ): Adjacent / Hypotenuse
  • Tangent (Tan θ): Opposite / Adjacent

Mnemonics for Trigonometric Ratios

Remember the acronyms to help with trigonometric ratios:

  • SOH: Sine = Opposite / Hypotenuse
  • CAH: Cosine = Adjacent / Hypotenuse
  • TOA: Tangent = Opposite / Adjacent

Examples

Example 1: Finding x and y

Given a right-angled triangle with a 30° angle, the side opposite is 4, the adjacent side is x, and the hypotenuse is y. Find x and y.

To find x, we can use the tangent ratio:

Tan 30° = Opposite / Adjacent = 4 / x

x * Tan 30° = 4

x = 4 / Tan 30°

x ≈ 4 / 0.5774

x ≈ 6.93

To find y, we can use the sine ratio:

Sin 30° = Opposite / Hypotenuse = 4 / y

y * Sin 30° = 4

y = 4 / Sin 30°

y = 4 / 0.5

y = 8

Example 2: Finding x (Triangle B)

Given a right-angled triangle B with a 30° angle and the opposite side equal to 13. Find the hypotenuse (x).

Solution:

Sin 30° = Opposite / Hypotenuse = 13 / x

x * Sin 30° = 13

x = 13 / Sin 30°

x = 13 / 0.5

x = 26

Example 3: Finding x (Triangle C)

Given a right-angled triangle C with a 75° angle and the adjacent side equal to 5. Find the opposite side (x).

Solution:

Tan 75° = Opposite / Adjacent = x / 5

x = 5 * Tan 75°

x ≈ 5 * 3.732

x ≈ 18.66
