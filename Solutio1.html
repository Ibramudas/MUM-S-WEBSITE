SOLUTION TO EXERCISE (1)

  1. b² - 14b + 45

    Solution:

    b² - 9b - 5b + 45

    (b² - 9b) - (5b - 45)

    b(b - 9) - 5(b - 9)

    (b - 5)(b - 9)

  2. 6d² - 7d - 20

    Solution:

    6d² - 15d + 8d - 20

    (6d² - 15d) + (8d - 20)

    3d(2d - 5) + 4(2d - 5)

    (3d + 4)(2d - 5)

  3. 2c² + 8c - 24

    Solution:

    2c² + 12c - 4c - 24

    (2c² + 12c) - (4c + 24)

    2c(c + 6) - 4(c + 6)

    (2c - 4)(c + 6)

EXERCISE 2

  1. Solve: 2x² - x - 21 = 0

    Solution:

    2x² - 7x + 6x - 21 = 0

    (2x² + 6x) - (7x + 21) = 0

    2x(x + 3) - 7(x + 3) = 0

    (2x - 7)(x + 3) = 0

    So, either:

    2x - 7 = 0 OR x + 3 = 0

    2x = 7 OR x = -3

    x = 7/2 OR x = -3

  2. Solve: 2p² + 11p = 30

    Solution:

    First, rearrange the equation:

    2p² + 11p - 30 = 0

    Now, factorise:

    2p² + 15p - 4p - 30 = 0

    (2p² - 4p) + (15p - 30) = 0

    2p(p - 2) + 15(p - 2) = 0

    (2p + 15)(p - 2) = 0

    So, either:

    2p + 15 = 0 OR p - 2 = 0

    2p = -15 OR p = 2

    p = -15/2 OR p = 2
