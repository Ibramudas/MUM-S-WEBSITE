Review of Expansion and Factorisation

Recall: Expansion of algebraic expression has to do with removing brackets and simplifying.

While factorisation of an algebraic expression means expressing it as a product of factors.

Examples: Expansion

  1. (r - x)(r + y) = r(r + y) - x(r + y) = r² + ry - xr - xy
  2. (x - 5)(x + 8) = x(x + 8) - 5(x + 8) 
                   = x² + 8x - 5x - 40 = x² + 3x - 40
  3. (m + 4n)² = (m + 4n)(m + 4n)
                   = m(m + 4n) + 4n(m + 4n) 
                   = m² + 4mn + 4mn + 16n² = m² + 8mn + 16n²

Examples: Factorisation

  1. Factorise 8p - 20q 
            = 4(8p/4 - 20q/4) 
            = 4(2p - 5q)
  2. Factorise 12x² + 3x - 4x - 1 
            (12x² + 3x) - (4x + 1) 
            = 3x(4x + 1) - 1(4x + 1)
             = (3x - 1)(4x + 1)
  3. Factorise 3x - 2dy + 3y - 2dx 
            Group terms: (3x + 3y) - (2dy + 2dx)
             = 3(x + y) - 2d(y + x)
             = (3 - 2d)(x + y)

Difference of Two Squares

(a - b)(a + b) = a² + ab - ab - b² = a² - b² 
           Hence, a² - b² = (a + b)(a - b)
:

Examples: Difference of Two Squares

  1. y² - 4 = (y)² - (2)² 
                 = (y - 2)(y + 2)
  2. 36 - 9a² = 6² - (3a)² 
                 = (6 - 3a)(6 + 3a)
  3. Factorise: 5a² - 45 = 5(a² - 9) 
                 = 5[(a - 3)(a + 3)]
